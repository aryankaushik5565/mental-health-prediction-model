# 🧠 Mental Health Prediction System

## 📌 Overview

This project focuses on predicting mental health conditions using machine learning techniques on a real-world dataset. Unlike ideal datasets, mental health data is often noisy, subjective, and imbalanced, making this problem more challenging and realistic.

---

## 🎯 Problem Statement

The goal of this project is to build a predictive model that can classify whether an individual is likely to face mental health issues based on various input features.

---

## 📊 Results

* Achieved approximately **60% accuracy** on the test dataset
* Performance reflects the **complex and noisy nature of mental health data**
* Demonstrates challenges like **class imbalance and subjective inputs**
